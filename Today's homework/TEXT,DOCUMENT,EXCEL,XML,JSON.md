Difference between TEXT, DOCUMENT, EXCEL, XML, JSON.?



1\. TEXT ( .txt ):



A plain text file containing raw text with no formatting.

Features:

* No bold, colors, tables, images
* Only characters like words, numbers, symbols
* Very lightweight

Where used:

* Notes
* Logs
* Simple data storing



2\. DOCUMENT ( .doc / .docx / .pdf ):

Used for formatted text.

Features:

* Bold, italic, underline
* Images
* Tables
* Headings \& styles

Examples:

* MS Word .docx
* Google Docs
* PDF files (formatted text)

Where used:

* Resumes
* Reports
* Letters
* Official documents



3\. EXCEL ( .xls / .xlsx ):

A spreadsheet file used for structured rows and columns of data.

Features:

* Tables
* Formulas
* Charts
* Filters
* Data analysis

Where used:

* Data entry
* Finance
* Calculations
* Storing structured data



4\. XML ( .xml ):



A markup language used to store data in a tag-based hierarchical structure.

Features:

* Data stored inside custom tags
* Human-readable
* Machine-readable
* Good for configuration and data exchange

Example:

<student>

&nbsp;   <name>Sindhu</name>

&nbsp;   <age>22</age>

</student>

Where used:

* Web services (SOAP)
* App configurations
* RSS feeds
