Difference between NumPy, Pandas, Matplotlib, Seaborn.?



1\. NumPy:



A Python library for numerical and mathematical operations.

Used for:

* Fast calculations
* Working with arrays
* Linear algebra
* Matrices
* Scientific computing

Example:

import numpy as np

arr = np.array(\[1, 2, 3])



2\. Pandas:



A Python library for data manipulation and data analysis.

Used for:

* Handling tables (rows \& columns)
* Cleaning data
* Filtering, sorting
* Reading CSV/Excel files

Example:

import pandas as pd

df = pd.read\_csv("data.csv")



3\. Matplotlib:



A basic plotting library for visualizing data.

Used for:

* Line graphs
* Bar charts
* Scatter plots
* Histograms

Example:

import matplotlib.pyplot as plt

plt.plot(\[1, 2, 3])

plt.show()



4\. Seaborn:



A data visualization library built on top of Matplotlib.

Used for:

* Beautiful statistical charts
* Heatmaps
* Boxplots
* Pairplots
* Distribution plots

Example:

import seaborn as sns

sns.histplot(\[1, 2, 3, 4])
