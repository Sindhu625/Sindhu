What is FastAPI?



FastAPI is a modern, fast (high-performance) web framework for building APIs in Python.



It is used to create backend applications, REST APIs, and microservices.



* It is built on:



1.ASGI (Asynchronous Server Gateway Interface)



2.Starlette → for routing, requests, responses



2\.Pydantic → for data validation



Why FastAPI is Popular?



1.Very Fast Performance



2.Automatic Validation



3.Auto API Documentation



4.Easy to Write \& Read



5.Built in Async Support



6.Used in Production by Big Companies



7.Great for ML/AI \& Backend
